Обозначения входных параметров аналитической модели


№ Обозначение Описание
1 N Число рабочих станций сети
2 T0 Среднее значение времени дообработки на рабочей станции сети запроса от этой станции к базе данных на сервере
3 Тp Среднее значение времени формирования запроса от рабочей станции сети к базе данных на сервере
4 tk Среднее значение времени передачи запроса по каналу
5 C Число процессоров сервера
6 tпр Среднее значение времени обработки запроса в ЦП сервера
7 tдi Среднее значение времени обработки запроса в диске сервера
8 m Число дисков
9 γ Вероятность обращения запроса к ЦП после обработки на диске. Обслуживание заявок во всех ОА подчиняется экспоненциальному закону
10 Δ Параметр регулирования точности вычисления
11 Nзн Количество знаков после запятой
